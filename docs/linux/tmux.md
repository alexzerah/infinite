# Tmux

```shell
tmux
```

Par default `tmux` utilise les commandes suivantes :

## Split les screens

- `ctrl + b` puis `%` : Split vertical
- `ctrl + b` puis `"` : Spllit horizontal

## Se deplacer

- `ctrl + b` puis fleches (up, down, left, right) : Se deplacer entre les fenetres

## Exit

- `exit` : Quitte le terminal
